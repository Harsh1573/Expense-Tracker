#  Expense Tracker Web App

A simple and responsive Expense Tracker web application built using **HTML, CSS, and JavaScript**.  
It helps users manage their income and expenses efficiently with real-time updates.

---

##  Features

- Add and delete transactions
- Real-time balance calculation
- Separate tracking of income and expenses
- Data stored in browser using LocalStorage
- Responsive and clean UI design
- Automatic updates on every transaction

---

##  Tech Stack

- HTML5
- CSS3
- JavaScript (DOM Manipulation & LocalStorage)
